兩百字內的個人簡介

WeHelp的各位大家好，我是李家誠，畢業於輔仁大學統計資訊學系。 畢業後有兩年的工作經驗。兩份的工作職位都是業務， 在上一份軟體服務業中因為與工程師協作，而接觸到程式方面的知識， 而想起大學曾經修過的程式設計課程，但隨著年日已過，許多知識已經淡忘， 又藉著工作的原因再次接觸到，從而產生學習的興趣，想要藉由這次機會轉職成為軟體工程師。

為了成為軟體工程師，做過什麼努力？若有具體作品請分享給我們。

在上一份工作中有考取Google Cloud Platform雲端架構師的證照， 也是因為在準備的過程中，重新產生了學習的動力與興趣，所以想要轉職成軟體工程師。

如果參與這個訓練，會怎麼安排學習時間？

因為離職後時間自由的關係，同時有安排想要考研究所的目標， 已經有三個月的時間每天早上到圖書館報到，學習軟體、硬體相關課程， 如果參加這個訓練，會安排每天的八個小時的讀書時間，大量接觸程式的知識。

是否有想要加入的公司？為什麼想加入該公司？

我想對於每個幻想踏入軟體業的人（例如我）心中目標的公司應該都是Google。 但對於目前的我來說，我希望在接受完訓練後，可以進到一個具有規模的科技公司， 在我有興趣的部門，繼續在軟體工程師的職涯上學習。

請描述一件產生明顯負面情緒的經歷，如何處理該情緒？

其實我是比較樂觀的人，但是在離職後偶爾還是會有情緒低落的時候，害怕自己無法成功， 在這個時候我會跟身邊比較親近的人聊聊，也會跟神禱告，讓負面情緒比較快能好轉。

關於這份申請網頁，分享一個開發時的技術心得。

對於我來說這是我第一次開發自己的網頁，一切都感到很特別， 總覺得跟大學上課的時候不一樣，是真的要為了自己的能力去學習一個技能， 數入程式碼後能執行出來很有成就感，有很多語法可以套用讓網頁關看起來更舒服。

請用簡短兩三句話，描述對你而言，最重要的一項人際交往原則。

彼此幫助，不管是物質上、情緒上，都能帶給彼此好的影響。

其他想要對我們說的事情？

在YouTube首頁上剛好刷到你們的影片，看到許多人成功轉職的成果，感到非常厲害跟羨慕， 也十分欣賞訓練營的整個申請流程，讓自己更確信自己的目標， 祝福WeHelp的各位老師，在職涯上也能順順利利。
